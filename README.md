# 🌍 Complete Agricultural Research Project

## About This Project
A complete end-to-end agricultural research project analyzing crop yield, rainfall patterns, fertilizer use and market prices for 8 Nigerian crops.

## Tools Used
- Python | Pandas | Matplotlib | Google Colab

## Key Findings
- Cassava highest yielding crop at 4000kg
- Rainfall-yield correlation of 0.97
- Complete 4-chart research dashboard

## Crops Analyzed
Maize | Rice | Bean | Tomato | Yam | Soybean | Cassava | Groundnut
